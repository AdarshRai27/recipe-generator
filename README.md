# recipe-generator
# 🍽️ AI Recipe Generator

This project generates cooking recipes using AI based on user-provided ingredients.

## 🚀 Features

* Recipe generation using OpenAI GPT models
* Text preprocessing using NLTK
* CLI-based interaction
* Dataset preprocessing support



## ⚙️ Installation

```bash
git clone https://github.com/yourusername/recipe-generator.git
cd recipe-generator
pip install -r requirements.txt
```

## 🔑 Setup API Key

```bash
export OPENAI_API_KEY="your_api_key"
```

## ▶️ Run

```bash
python main.py
```

## 🧠 Example

Input:

```
chicken, rice, garlic
```

Output:

```
Delicious garlic chicken rice recipe...
```
